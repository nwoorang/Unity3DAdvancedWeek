# Unity3D Advanced Week - 인벤토리 기능 구현

유니티 3D에서 인벤토리 시스템을 구현해보는 실습형 프로젝트입니다.  
고급 기능 위주로 직접 구현해보고 테스트할 수 있도록 구성

## 📌 주요 기능

- 아이템 획득 및 삭제
- 인벤토리 UI 표시 / 닫기
- 아이템 드래그 앤 드롭
- 아이템 설명 툴팁
- 슬롯 자동 정렬
